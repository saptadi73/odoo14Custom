Shared Master Module

GRT Business Category Base

Central module for shared Business Category management across CRM, Sales, Purchase, Expense, and Inventory. Create the category once, then reuse it everywhere.

What This Module Owns

  • Master model crm.business.category
  • Shared analytic account on category level
  • User access setup per business category
  • Reusable default/mixin behavior for other modules

Where To Configure

  • Settings > Administration > Business Categories
  • Settings > Administration > User Category Access
  • Settings > Users & Companies > Users then open the Business Category Access tab

Access Logic

  • Allowed: manual permission
  • Team: inherited from team membership
  • Effective: final usable categories
  • Default: default category when creating records
  • Active: current user working context

Recommended Setup Order

  1. Create the master business categories.
  2. Set the correct company on each category.
  3. Open the user form.
  4. Fill Allowed Business Categories.
  5. Use Sync Team Access if team-based access is needed.
  6. Confirm the category appears in Effective Business Categories.
  7. Select Default Business Category.
  8. Select Active Business Category.

If a category does not appear in the Default or Active dropdown, it usually means the category is not yet included in the user's Allowed Business Categories, or the category company does not match the user's companies.
